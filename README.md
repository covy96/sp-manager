# SP Manager

Gestionale web per studi di architettura costruito con React, Vite e Supabase.

## Stack

- Vite + React
- Supabase client (`@supabase/supabase-js`)
- React Router
- Tailwind CSS

## Setup

1. Installa le dipendenze:

   ```bash
   npm install
   ```

2. Avvia il server di sviluppo:

   ```bash
   npm run dev
   ```

3. Apri l'URL mostrato nel terminale.
